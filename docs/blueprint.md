# **App Name**: laSalapp

## Core Features:

- Google Authentication with Domain Restriction: Secure access to the app using Google Authentication, restricted to users with the @atresquarts.com domain.
- Team Management: Add and manage team members with different roles and permissions.
- Programming Management: Track shows and companies interested in performing at laSala. Record all interactions.
- Calendar and Scheduling: Manage event schedules, including confirmed performances and planning for marketing activities like social media posts and ticket sales.
- Expense Tracking: Record expenses made by team members, structural costs, materials, and basic expenses, with amortization tracking.
- Meeting Minutes Management: Record meeting discussions and decisions, searchable by keywords.
- Responsibilities Dashboard with AI Tooling: Track team responsibilities with deadlines, assignees, and a checklist. The AI tool analyzes the current team workload and uses reasoning to decide who to assign new responsibilities to, in order to avoid over-burdening a particular team member.
- General To-Do List: Create and manage a general to-do list for tasks that need to be completed, tracking progress and deadlines.
- In-House Production Hub: Create private pages for in-house productions where you can invite external collaborators with view-only access, linking to assets stored on Google Drive (scripts, songs, etc.).
- Idea Brainstorming: Dedicated space to record and organize new ideas from brainstorm sessions.

## Style Guidelines:

- Primary color: Pale Yellow (#FFF4A4) for a warm, inviting feel.
- Background color: Very light Yellow (#FFFBE6) to provide a soft, clean backdrop.
- Accent color: Soft Green (#75EAC3) to highlight interactive elements.
- Body font: 'PT Sans', a humanist sans-serif, suitable for both headlines and body text
- Use clean, minimalist icons to represent different sections and actions within the app.
- Prioritize a mobile-first design, ensuring the app is fully responsive and easy to use on smaller screens before scaling up to desktop.
- Use subtle transitions and animations to provide feedback and enhance the user experience without being distracting.
